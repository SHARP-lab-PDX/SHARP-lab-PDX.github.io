# SHARP Lab Website

Last dev update: Oct 5, 2025

Domain: www.sharplabpdx.com \
Site generator: Jekyll 4.3 \
CMS: Decap 3.0 \
Deployed with: Netlify \
Package list: Gemfile \

## Installing the project

`bundle install`

## Running the dev server

`jekyll serve` \
Local server: http://127.0.0.1:4000/

## Build testing

`jekyll build`

## Dependencies:

- jekyll-timeago
- Bootstrap 5.3
- Academic Icons CDN
